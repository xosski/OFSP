CONFIGURATION GUIDE
================================================================================

1. OPTIONAL: Set API Keys for Premium LLM Providers

   Environment Variables:
   - MISTRAL_API_KEY=your_key_here
   - OPENAI_API_KEY=your_key_here
   - AZURE_OPENAI_API_KEY=your_key_here
   - AZURE_OPENAI_ENDPOINT=https://...

   OR Edit .hades_config.json:
   {
     "mistral_api_key": "your_key_here",
     "openai_api_key": "your_key_here",
     "ai_provider": "mistral"
   }

2. OPTIONAL: Enable Local LLM (Ollama)
   
   Install: https://ollama.ai
   Pull model: ollama pull mistral
   Run: ollama serve
   Auto-detected at: http://localhost:11434

3. DEFAULT: Always uses Fallback Provider
   
   Rule-based provider always available
   No keys required
   Works offline
   Provides basic analysis
