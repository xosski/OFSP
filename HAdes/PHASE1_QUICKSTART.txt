TROUBLESHOOTING
================================================================================

PROBLEM 1: Tabs don't appear after starting HadesAI
────────────────────────────────────
Cause: Import failed
Solution:
1. Check console for error messages
2. Verify files exist:
   ls -la modules/obsidian_core_integration.py
   ls -la modules/ethical_control_integration.py
   ls -la modules/malware_engine_integration.py
   ls -la phase1_gui_tabs.py
3. Run test suite: python test_phase1_integration.py
4. Restart HadesAI.py

PROBLEM 2: "All operations show Unauthorized"
────────────────────────────────────
Cause: Environment not authorized
Solution:
1. Set environment: set ENV_NAME=redteam-lab
2. Restart HadesAI
3. Check 🔒 Ethical Control tab status
4. Add target to whitelist before using

PROBLEM 3: Payload mutation fails
────────────────────────────────────
Cause: Invalid input
Solution:
1. Ensure payload input is not empty
2. Try with a simple test: print("hello")
3. Check method selection is correct
4. Look at console for error details

PROBLEM 4: Detection probability very high (>90%)
────────────────────────────────────
Cause: Payload has many suspicious indicators
Solution:
1. Use polymorphic variants instead
2. Add anti-analysis wrapper
3. Use different mutation method
4. Split payload into stages

PROBLEM 5: Tab crashes with error
────────────────────────────────────
Cause: UI error
Solution:
1. Close tab and reopen
2. Check console for stack trace
3. Simplify input (fewer iterations, shorter payload)
4. Restart HadesAI
