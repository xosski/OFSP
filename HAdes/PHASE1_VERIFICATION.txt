COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE 1: "ImportError: cannot import name 'ObsidianCoreTab'"
SOLUTION: 
- Verify phase1_gui_tabs.py exists
- Check file is in correct directory
- Run: python -c "from phase1_gui_tabs import ObsidianCoreTab"

ISSUE 2: "Tabs don't appear after starting HadesAI"
SOLUTION:
- Check HAS_PHASE1_INTEGRATION = True in logs
- Run test suite: python test_phase1_integration.py
- Verify HadesAI.py has correct imports (lines 145-156)

ISSUE 3: "Test suite fails with 'ModuleNotFoundError'"
SOLUTION:
- Verify all 3 modules exist in modules/ directory
- Check Python version is 3.7+
- Run: python -c "from modules.obsidian_core_integration import get_obsidian_core"

ISSUE 4: "Authorization always shows 'Unauthorized'"
SOLUTION:
- Set environment: set ENV_NAME=redteam-lab
- Restart HadesAI.py
- Check Ethical Control tab status

ISSUE 5: "Payload mutation fails"
SOLUTION:
- Ensure payload input is not empty
- Try with simple test: print("hello")
- Check selected method is valid
- Look at console for error details

For more help, see PHASE1_DEPLOYMENT_GUIDE.md TROUBLESHOOTING section.
