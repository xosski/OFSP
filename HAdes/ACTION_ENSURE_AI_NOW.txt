TROUBLESHOOTING:
================

If VERIFY_AI_USAGE.py shows failures:

- Failure on "Configuration"
  → Run: python setup_exploit_generator_ai.py

- Failure on "Providers"
  → Check .hades_config.json has mistral_api_key
  → Verify API key is not empty

- Failure on "AI Generator"
  → Make sure mistralai is installed: pip install mistralai
  → Check internet connection

- Failure on "AI Generation"
  → Verify Mistral API key is valid
  → Try: pip install --upgrade mistralai openai
