INTERACTIVE FEATURES:
====================

1. CLICK ON A THREAT:
   ├─ Details load in right panel
   ├─ All 4 tabs populate with code
   └─ Color highlights severity (RED=HIGH, YELLOW=MEDIUM, GREEN=LOW)

2. FILTER BY SEVERITY:
   ├─ "All" - shows all threats
   ├─ "HIGH" - only high severity
   ├─ "MEDIUM" - only medium severity
   └─ "LOW" - only low severity

3. LIMIT RESULTS:
   ├─ Default: 100
   ├─ Range: 10-1000
   └─ Shows "Showing X/Y threats" label

4. EXPORT FINDINGS:
   ├─ Creates cache_reports/ folder
   ├─ Saves JSON with full code
   └─ Saves HTML with highlighting

5. SCAN PROGRESS:
   ├─ Progress bar shows scan progress
   ├─ Status updates as threats found
   └─ "Stop" button available during scan


TYPICAL WORKFLOW:
=================

1. Click "Scan Browser Cache" button
   └─ Scan begins in background
      Status: "Initializing scanner..."

2. Threats start appearing in left panel
   ├─ Files are listed as they're found
   ├─ Progress bar advances
   └─ Status shows file count

3. Click on a threat to view details
   ├─ Summary tab shows metadata
   ├─ Code tab shows matched code
   ├─ Context tab shows surrounding code
   └─ Full Code tab shows entire file

4. Use Filter to find specific threats
   ├─ Filter for "HIGH" to find critical issues
   ├─ Filter for "MEDIUM" for secondary concerns
   └─ Filter for "LOW" for informational

5. Optionally export all findings
   ├─ Click "Export Findings"
   ├─ Creates JSON report (machine-readable)
   ├─ Creates HTML report (human-readable)
   └─ Both include full code content

6. Review results in chat
   ├─ Summary appears in chat window
   ├─ Statistics broken down by type/severity/browser
   └─ Links to tab for detailed viewing


COLOR CODING:
=============

RED (#ff6b6b)      = HIGH severity threats
YELLOW (#ffa94d)   = MEDIUM severity threats
GREEN (#69db7c)    = LOW severity threats

Column Colors Update When Severity Changes:
  File Name | Threat Type | SEVERITY | Browser | Size
  ├─ All columns highlighted with severity color
  └─ Makes visual scanning easier

TROUBLESHOOTING VISUAL GUIDE:
=============================

Problem: No threats showing after scan
Solution: Check chat for scan results, verify cache directories exist

Problem: Details not loading when clicking threat
Solution: Try clicking different threat, restart tab

Problem: Full Code is truncated
Solution: Tab shows first 50KB, export to file for complete code

Problem: Code tab shows nothing
Solution: This threat may have no matched code, try other threats

Problem: Can't filter threats
Solution: Ensure filter dropdown is working, try clicking it again
