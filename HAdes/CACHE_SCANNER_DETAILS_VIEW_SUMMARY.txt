================================================================================
THE FOUR DETAIL TABS
================================================================================

TAB 1 - SUMMARY:
   Shows threat metadata
   - Type, Severity, Browser
   - File path & hash
   - Match position & length
   - Detection timestamp
   - Exploit link (if matched)

TAB 2 - CODE:
   Shows the matched malicious code
   - Exact code found
   - Character count

TAB 3 - CONTEXT:
   Shows code before, during, and after threat
   - 500B of code BEFORE threat
   - [THREAT CODE]
   - 500B of code AFTER threat

TAB 4 - FULL CODE:
   Shows complete file content
   - First 50KB of file
   - Or complete if smaller
   - Truncation indicator if needed

INTERACTIVE FEATURES

   - Details load in all 4 tabs on right

   - Switches directly to Code tab

   - Progress bar shows scan %
   - Status updates with file count
   - Stop button available

7. COLOR CODING
   - RED = HIGH severity
