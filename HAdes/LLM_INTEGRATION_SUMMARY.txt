================================================================================
TROUBLESHOOTING
================================================================================

Provider Not Showing?
    1. Install dependencies: pip install openai mistralai ollama
    2. Check environment variables
    3. Verify API keys are valid
    4. Run: python test_llm_integration.py

Slow Responses?
    1. Check API provider status
    2. Use local Ollama instead
    3. Reduce max_tokens in settings
    4. Check network latency

No LLM Available?
    1. Fallback provider always works
    2. Uses rule-based responses
    3. Automatic fallback if primary fails
    4. Run: python test_llm_integration.py

Cannot Save Conversations?
    1. Check database permissions
    2. Delete .db-shm and .db-wal files
    3. Restart application
    4. Check disk space
