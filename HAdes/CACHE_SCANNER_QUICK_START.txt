------------------------
from cache_scanner_enhanced import EnhancedCacheScanner

scanner = EnhancedCacheScanner()

results = scanner.scan_cache_with_details("path/to/file", "Chrome")

KEY IMPROVEMENTS:
=================

Code Visibility:
  Before: code_snippet (200 chars max)
  After:  full_code (500KB max) + context_before/after

Learned Exploits:
  Before: Manual queries, inconsistent results
  After:  Auto-loaded at startup, always available

Storage:
  Before: Lost after scan
  After:  Persisted in database with timestamps

Reporting:
  Before: Console output only
  After:  JSON + HTML with code highlighting

Database Schema:
  Old Table: threat_findings (simple)
  New Tables:
    - cache_detections (with full_code column)
    - learned_exploits (with better tracking)
    - code_patterns (for pattern analysis)

EXAMPLE USAGE:
==============

# Get full threat details
scanner = EnhancedCacheScanner()
detections = scanner.get_cache_detections(limit=10)

for d in detections:
    print(f"Path: {d['cache_path']}")
    print(f"Type: {d['threat_type']}")
    print(f"Severity: {d['severity']}")
    print(f"Full Code ({len(d['full_code'])} bytes):")
    print(d['full_code'])
    print(f"Matched Exploit ID: {d['matched_exploit_id']}")

# Get threat summary
summary = scanner.get_threat_summary()
print(f"Total threats: {summary['total_detections']}")
print(f"By type: {summary['by_threat_type']}")

# Export with full code
scanner.export_findings_to_json("findings.json")
scanner.export_findings_to_html("findings.html")
