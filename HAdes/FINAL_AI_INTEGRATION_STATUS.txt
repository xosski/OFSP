TROUBLESHOOTING REFERENCE
================================================================================

Issue: Still seeing placeholder text
Solution:
  1. Run VERIFY_AI_USAGE.py - should all pass
  2. Check .hades_config.json has mistral_api_key
  3. Check console logs for errors
  4. Restart HadesAI

Issue: "No AI providers configured"
Solution:
  1. Verify .hades_config.json exists
  2. Run python setup_exploit_generator_ai.py
  3. Make sure API key is in config file

Issue: "AI-generated: False" in logs
Solution:
  1. Check internet connection
  2. Verify Mistral API key is valid
  3. Try pip install --upgrade mistralai
  4. Check Mistral API status

Issue: Error calling AI
Solution:
  1. Test API key directly:
     python -c "from mistralai import Mistral; ..."
  2. Check firewall/VPN blocking API calls
  3. Verify Python packages are up to date
